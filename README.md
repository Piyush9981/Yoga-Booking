# Yoga Booking Website
A web-based yoga session booking system built using Flask. 
Users can browse yoga sessions, book events, and manage their bookings.

## Features
- User registration and login
- View upcoming yoga sessions and previous sessions details 
- Book yoga events
- View booking history

## Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
